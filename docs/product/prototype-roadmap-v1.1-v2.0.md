## v1.1 — 改造起点

> **焦点：编辑器改造 + 性能基建**

### 布局变更

```
┌────────────────────────────────────────────────────────────┐
│ [← 书架]  织梦机 v1.1           [文档] [画板] [设定] [判断] │  ← 导航栏
├──────────────────────┬────────────────────────────────────┤
│                      │                                    │
│  左侧栏 (新增)       │    主内容区                          │
│  ┌──────────────┐    │    ┌──────────────────────────┐    │
│  │ 📖 大纲树     │    │    │  富文本编辑器 (Tiptap)    │    │
│  │  ├ 第一章     │    │    │  ┌────────────────────┐  │    │
│  │  ├ 第二章     │    │    │  │                    │  │    │
│  │  │  ├ 场景A   │    │    │  │  WikiLink 支持     │  │    │
│  │  │  └ 场景B   │    │    │  │  [[角色名]] 自动   │  │    │
│  │  └ 第三章     │    │    │  │  链接到对象       │  │    │
│  │              │    │    │  │                    │  │    │
│  │  📦 引用面板  │    │    │  └────────────────────┘  │    │
│  │  当前对象被   │    │    │                            │    │
│  │  N 处引用     │    │    └──────────────────────────┘    │
│  └──────────────┘    │                                    │
│                      ├────────────────────────────────────┤
│                      │  底部状态栏 (新增)                  │
│                      │  对象数: 12  |  连线: 8  |  保存中  │
├──────────────────────┴────────────────────────────────────┤
│  Inspector (右侧栏，已有)                                   │
└────────────────────────────────────────────────────────────┘
```

### 新增 UI 元素

| 元素 | 位置 | 说明 |
|------|------|------|
| 🆕 左侧大纲树 | 左侧栏 | 按结构层级展示所有对象，可拖拽排序 |
| 🆕 引用面板 | 左侧栏下方 | 显示当前对象的被引用计数和引用列表 |
| 🆕 底部状态栏 | 底部 | 实时显示对象数、连线数、保存状态 |
| 🆕 WikiLink 语法高亮 | 编辑器内 | `[[名称]]` 自动解析为对象链接，点击跳转 |

### 交互变化

- 编辑器升级为 Tiptap 全功能富文本编辑器
- 大纲树 `click` 跳转对象，`drag` 调整层级
- WikiLink 自动补全：输入 `[[` 触发对象名称选择器
- 新增自动保存指示器（草稿图标 ↔ 已保存图标）

### 移除/隐藏

- (无)

### 与 v1.0 对比

| 维度 | v1.0 (当前) | v1.1 |
|------|------------|------|
| 编辑器 | 基本文本输入 | Tiptap 富文本 + WikiLink |
| 导航 | 标签栏切换 | 标签栏 + 大纲树双导航 |
| 状态反馈 | 无 | 底部状态栏 |
| 引用关系 | 不可见 | 引用面板可见 |

### 新增 UI 元素

| 元素 | 位置 | 说明 |
|------|------|------|
| 🆕 AI 助手浮窗 | 左侧栏底部 | 折叠/展开的 AI 对话入口，初始折叠 |
| 🆕 Key 管理设置页 | 设置面板 | API Key 输入、加密存储、连接测试 |
| 🆕 设置入口 ⚙️ | 导航栏右侧 | 点击打开设置面板 |
| 🆕 连接状态指示器 | 状态栏 | AI 服务连接状态（已连接/未连接/错误） |

- 点击 AI 助手浮窗展开 → 底部弹出对话面板（不是全屏）
- Key 配置页面: 输入 → 加密存储 → 连接测试 → 保存
- 连接状态实时反映在状态栏

### 与 v1.1 对比

| 维度 | v1.1 | v1.2 |
|------|------|------|
| AI 能力 | 无 | 基础 AI 助手入口（需 Key）|
| 设置 | 无独立设置页 | ⚙️ 设置面板 + Key 管理 |
| 状态栏 | 对象/连线/保存 | + 连接状态指示器 |

### 新增 UI 元素

| 元素 | 位置 | 说明 |
|------|------|------|
| 🆕 协作者列表 | 左侧栏大纲树上方 | 显示当前参与的 Agent 和用户 |
| 🆕 协作光标 | 编辑器内 | 每个协作者有颜色标记的光标位置 |
| 🆕 AI 正在输入指示器 | 编辑器顶部 | 显示当前正在写作的 Agent 名称 |
| 🆕 协作工具栏 | 编辑器上方 | 邀请/暂停/查看变更 |
| 🆕 变更历史面板 | 右侧新增 | 按时间线显示所有 AI 协作修改记录 |
| 🆕 接受/拒绝按钮 | 变更历史 | 逐条接受或拒绝 AI 的修改建议 |

### 交互变化

- AI Agent 可以直接在编辑器中写入内容（非覆盖模式）
- 用户可以看到 AI 的实时打字效果（类似 Google Docs 协作者）
- 每个 AI 修改可追溯、可接受/拒绝
- 可随时暂停所有 AI 协作，进入手动编辑模式

### 与 v1.4 对比

| 维度 | v1.4 | v1.5 |
|------|------|------|
| AI 产出 | 后台执行，结果呈现在面板 | 直接写入编辑器，实时可见 |
| 协作模式 | 任务式（提交→等待→查看） | 实时协作（像 Google Docs）|
| 修改控制 | 手动复制 | 逐条接受/拒绝 + 历史追溯 |
| 用户体验 | 分离感 | 融合感 |
